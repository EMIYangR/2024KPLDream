# 2024KPLDream

本项目基于Python和JavaScript，运行程序即可将2024年KPL梦之队的游戏端投票数据加载并写入到示例Excel中，你可以自定义值来自行选择你要写入的数据。

## 默认值：

### 选手

|  值   |  对应值  |
| :---: | :------: |
| nick  |    ID    |
| road  |   分路   |
| vote  |   票数   |
| tname | 所属队伍 |

### 教练

|    值    |   对应值   |
| :------: | :--------: |
|   name   | ID（姓名） |
| votevote |    票数    |

## 可选值

### 选手

| :---: | :------: |
